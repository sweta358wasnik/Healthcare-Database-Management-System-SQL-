

# 🏥 Healthcare Database Managemnet System

## 📌 Project Overview

This project is based on a **Healthcare Database Management System** using SQL. It focuses on analyzing data related to patients, doctors, appointments, billing, and prescriptions.

The project demonstrates real-world SQL operations such as data retrieval, filtering, joins, aggregations, and reporting insights.

---

## 📂 Database Tables Used

* **Patients**
* **Doctors**
* **Appointments**
* **Billing**
* **Prescriptions**

---

## 🚀 Key Features & Analysis

*  Retrieve patient and appointment details
*  Analyze billing status (Paid / Pending)
*  Calculate total billed and total paid amount
*  Doctor-wise appointment analysis
*  Identify latest appointments of patients
*  Most common reasons for appointments
*  Patient demographics analysis (Gender-wise)
*  Find patients with no appointments
*  Identify appointments without billing

---

## 🧠 SQL Concepts Used

* SELECT statements
* WHERE conditions
* INNER JOIN / LEFT JOIN
* GROUP BY & ORDER BY
* Aggregate Functions (SUM, COUNT, AVG, MAX)
* Subqueries
* Date Functions

---

## 🛠️ Tools & Technologies
